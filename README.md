#### Works with
* Almost any linux based shell.
* Cygwin shell on Windows.
* Git Bash on Windows.

#### Requirements
* cURL
* Some patience.

#### Setup Instructions
* If you are on linux, ensure that `gitbackup.sh` is chmodded to `755`.
* Run `./gitbackup.sh`.
